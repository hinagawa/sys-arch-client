## Use python >= 3.7

## Install poetry
```shell
pip install poetry
```

## Django

#### Install the project dependencies
```shell
cd server && poetry install 
```

#### Spawn a shell within the virtual environment
```shell
poetry shell
```

#### Start server
```shell
cd src && python manage.py runserver
```

## Client

#### Install the project dependencies
```shell
cd client && poetry install 
```

#### Spawn a shell within the virtual environment
```shell
poetry shell
```

#### Start client
```shell
python main.py
```
